# 📊 Employee Salary Analysis Dashboard

## Project Overview
This Power BI project analyzes employee salary data to understand salary distribution, department performance, and employee trends through interactive dashboards.

## Objectives
- Analyze salary distribution
- Compare department-wise salaries
- Track employee performance metrics
- Generate business insights

## Tools Used
- Power BI
- Excel
- Data Visualization

## Features
✅ Salary Distribution Analysis  
✅ Department Comparison  
✅ KPI Metrics  
✅ Interactive Dashboard  

## Dataset Fields
- Employee ID
- Department
- Salary
- Experience
- Performance Rating

## Key Insights
- Department salary comparison completed
- Salary trends identified
- Employee performance metrics analyzed
